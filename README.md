# 电商热销数据爬取分析系统

一个专业的电商平台热销数据爬取和分析工具，支持多平台数据采集、多维度统计分析和智能选品推荐。

## 功能特性

### 🛒 支持平台
- **TikTok Shop** - 热门商品和趋势数据
- **Amazon** - 畅销榜单和商品信息
- **Temu** - 热销商品和价格趋势
- **eBay** - 拍卖和固定价格商品数据

### 📊 数据分析维度
- **时间维度**: 15分钟、1小时、1天、1周、1月
- **品类分析**: 各类目热度排行和趋势
- **商品分析**: 热门商品识别和价格监控
- **地区分析**: 美区市场数据专项分析

### 🎯 选品功能
- 智能选品推荐算法
- 竞争度分析
- 利润空间计算
- 趋势预测

### 📈 可视化界面
- 实时数据仪表板
- 交互式图表分析
- 自定义报表生成
- 数据导出功能

## 技术架构

```
├── scrapers/          # 爬虫模块
├── database/          # 数据库模型
├── analysis/          # 数据分析
├── web/              # Web界面
├── scheduler/        # 定时任务
├── config/           # 配置文件
└── utils/            # 工具函数
```

## 快速开始

### 环境要求
- Python 3.8+
- Node.js 16+
- PostgreSQL/SQLite

### 安装依赖
```bash
pip install -r requirements.txt
npm install
```

### 运行系统
```bash
# 启动后端服务
python app.py

# 启动前端界面
npm run dev
```

## 使用说明

1. **配置爬虫参数**: 在 `config/settings.py` 中设置各平台的爬取参数
2. **启动数据采集**: 系统会自动按设定间隔采集数据
3. **查看分析结果**: 通过Web界面查看实时数据和分析报告
4. **导出数据**: 支持Excel、CSV等格式导出

## 注意事项

- 请遵守各平台的robots.txt规则
- 建议使用代理池避免IP封禁
